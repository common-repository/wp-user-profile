A WordPress plugin that makes Profile widget easily.

== Description ==

You can set up the author's profile with just a simple setup.

[Plugin page](http://e-joint.jp/works/wp-user-profile/)

著者のプロフィールを簡単な設定だけで設置することができます。

[プラグインページ](http://e-joint.jp/works/wp-user-profile/)

[Github](https://github.com/ejointjp/wp-user-profile)

== Installation ==

= Install =

1. Get "WP User Profile" from "Plugins > New add" on the WordPress dashboard.
2. Activate this.
3. Set it on the "Users > WP User Profile" in the dashboard.
4. In the "Users" in the dashboard, enter "Contact information" on the editing screen of each user.
5. If you want to customize the display, please read the above Plugin Page.

= インストール =

1. 「プラグイン > 新規追加」から「WP User Profile」を入手します。
2. プラグインを有効化します｡
3. ダッシュボードの「設定 > WP User Profile」で初期設定をします。
4. ダッシュボード内「ユーザー」内で、それぞれのユーザーの編集画面にある「連絡先情報」を入力します。
5. 表示のカスタマイズをしたい場合は、上記プラグインページをお読みください。
